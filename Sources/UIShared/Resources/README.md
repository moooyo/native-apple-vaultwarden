# EFF large wordlist

`eff_large_wordlist.txt` is the 7,776-word long Diceware list published by the
Electronic Frontier Foundation:

https://www.eff.org/files/2016/07/18/eff_large_wordlist.txt

EFF publishes the list under the Creative Commons Attribution 3.0 US license.
The five-digit Diceware indexes are retained for source verification; OpenVault
loads only the word column at runtime.
