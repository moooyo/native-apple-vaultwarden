# Fido2 Implementation Plan (M1 · Plan 6/N)

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Checkbox steps.

**Goal:** Build `Fido2` — the platform-agnostic **software WebAuthn authenticator** logic for passkeys: build `authenticatorData`, perform ES256 (P-256) assertion signing over `authenticatorData || clientDataHash`, generate P-256 credential keypairs, and produce registration `attestationObject` CBOR (`fmt = "none"`). This is the crypto/encoding core the (Xcode-gated) AutoFill extension will call; it is fully unit-testable on this CLT host via CryptoKit P256.

**Architecture:** SwiftPM library `Fido2` depending on `CryptoCore` (uses CryptoKit directly for `P256` + `SHA256`). Pure value types + stateless funcs. A minimal CBOR encoder (only what attestationObject/COSE need) lives here. Tests use the CLT executable convention (`swift run Fido2Tests`). ECDSA signatures are non-deterministic, so tests assert **sign-then-verify round-trips** + **exact structural bytes** (rpIdHash, flags, counter, CBOR shape), not exact signatures.

**Tech Stack:** Swift 6 (`swift-tools-version: 6.2`), CryptoKit (`P256`, `SHA256`), Foundation. Baseline iOS/macOS 26.

> **⚠️ TESTING CONVENTION** (same as prior plans): no XCTest; `.executableTarget` `Fido2Tests` via `swift run Fido2Tests`; copy `TestRunner`. TDD: check → fail → implement → pass → commit.

**Public API:**
```swift
public struct AuthenticatorFlags: OptionSet, Sendable {
    public let rawValue: UInt8
    public init(rawValue: UInt8)
    public static let userPresent      = AuthenticatorFlags(rawValue: 0x01) // UP
    public static let userVerified     = AuthenticatorFlags(rawValue: 0x04) // UV
    public static let attestedData     = AuthenticatorFlags(rawValue: 0x40) // AT
}

public struct CredentialKey: Sendable {
    public init()                                   // generate a fresh P-256 key
    public init(pkcs8 der: Data) throws             // import (the stored keyValue)
    public func exportPKCS8() -> Data               // for storage in fido2Credentials.keyValue
    public var publicKeyX963: Data { get }          // uncompressed point (0x04||X||Y)
}

public enum Fido2Error: Error, Equatable { case invalidKey, signFailed }

public enum Fido2Authenticator {
    /// authenticatorData = SHA256(rpId) || flags || signCount(4, big-endian) [|| attestedCredentialData]
    public static func authenticatorData(rpId: String, flags: AuthenticatorFlags,
                                         signCount: UInt32,
                                         attestedCredentialData: Data? = nil) -> Data

    /// WebAuthn assertion: returns DER ECDSA signature over (authenticatorData || clientDataHash).
    public static func assert(rpId: String, clientDataHash: Data, signCount: UInt32,
                              userVerified: Bool, key: CredentialKey) throws
        -> (authenticatorData: Data, signature: Data)

    /// WebAuthn registration with fmt="none": returns (attestationObject, credentialId, key).
    public static func register(rpId: String, clientDataHash: Data, credentialId: Data,
                                userVerified: Bool, aaguid: Data = Data(repeating: 0, count: 16))
        throws -> (attestationObject: Data, credentialKey: CredentialKey)
}
```

### Task 1: Package target + minimal CBOR encoder

**Files:** modify `Package.swift` (add `Fido2` lib dep `CryptoCore`; `Fido2Tests` exec); create `Sources/Fido2/CBOR.swift`, harness, `Checks_CBOR.swift`.

CBOR (RFC 8949) major types needed: 0 uint, 1 negative int, 2 byte string, 3 text string, 4 array, 5 map. Definite-length only. Map keys for COSE include negative ints.

- [ ] **Step 1: Failing checks** (known CBOR encodings):
  - uint `0` → `00`; `23` → `17`; `24` → `1818`; `255` → `18ff`; `256`→`190100`; `1000000`→`1a000f4240`.
  - negative `-1` → `20`; `-7` → `26` (ES256 label).
  - byte string `0x01 0x02` → `42 0102`; text `"a"` → `61 61`.
  - array `[1,2]` → `82 01 02`; map `{1:2,3:4}` → `a2 01 02 03 04`.
- [ ] **Step 2: Run** `swift run Fido2Tests` → FAIL.
- [ ] **Step 3: Implement** `CBOR.swift` — a `CBOR` enum value model + `encode() -> Data`, or direct builder funcs. Encode the initial byte as `(major << 5) | argument` with the 24/25/26/27 length forms:
```swift
import Foundation
public enum CBOR {
    case uint(UInt64), negint(UInt64) /* value = -1 - n */, bytes(Data), text(String)
    case array([CBOR]), map([(CBOR, CBOR)])

    public func encoded() -> Data {
        switch self {
        case .uint(let n): return CBOR.head(0, n)
        case .negint(let n): return CBOR.head(1, n)              // caller passes (-1 - value)
        case .bytes(let d): return CBOR.head(2, UInt64(d.count)) + d
        case .text(let s): let d = Data(s.utf8); return CBOR.head(3, UInt64(d.count)) + d
        case .array(let a): return a.reduce(CBOR.head(4, UInt64(a.count))) { $0 + $1.encoded() }
        case .map(let m): return m.reduce(CBOR.head(5, UInt64(m.count))) { $0 + $1.0.encoded() + $1.1.encoded() }
        }
    }
    public static func int(_ v: Int) -> CBOR { v >= 0 ? .uint(UInt64(v)) : .negint(UInt64(-1 - v)) }

    private static func head(_ major: UInt8, _ n: UInt64) -> Data {
        let m = major << 5
        switch n {
        case 0...23: return Data([m | UInt8(n)])
        case 24...0xff: return Data([m | 24, UInt8(n)])
        case 0x100...0xffff: return Data([m | 25]) + be(n, 2)
        case 0x10000...0xffff_ffff: return Data([m | 26]) + be(n, 4)
        default: return Data([m | 27]) + be(n, 8)
        }
    }
    private static func be(_ n: UInt64, _ bytes: Int) -> Data {
        Data((0..<bytes).reversed().map { UInt8((n >> (8 * $0)) & 0xff) })
    }
}
```
- [ ] **Step 4: Run** → pass. **Step 5: Commit** `feat(fido2): add minimal CBOR encoder`.

```swift
import Foundation
import CryptoKit
public struct CredentialKey: Sendable {
    private let priv: P256.Signing.PrivateKey
    public init() { priv = P256.Signing.PrivateKey() }
    public init(pkcs8 der: Data) throws {
        do { priv = try P256.Signing.PrivateKey(pkcs8DERRepresentation: der) }
        catch { throw Fido2Error.invalidKey }
    }
    public init(x963 der: Data) throws {                 // alt import if needed
        do { priv = try P256.Signing.PrivateKey(x963Representation: der) }
        catch { throw Fido2Error.invalidKey }
    }
    public func exportPKCS8() -> Data { priv.pkcs8DERRepresentation }
    public var publicKeyX963: Data { priv.publicKey.x963Representation }  // 0x04||X||Y (65 bytes)
    func sign(_ data: Data) throws -> Data {
        do { return try priv.signature(for: data).derRepresentation }     // WebAuthn wants ASN.1 DER
        catch { throw Fido2Error.signFailed }
    }
    var publicKey: P256.Signing.PublicKey { priv.publicKey }
}
```

```swift
import Foundation
public enum COSEKey {
    /// COSE_Key for an EC2 / ES256 / P-256 public key from an X9.63 uncompressed point.
    public static func encode(publicKeyX963 point: Data) -> Data {
        precondition(point.count == 65 && point[point.startIndex] == 0x04)
        let x = point.subdata(in: point.index(point.startIndex, offsetBy: 1)..<point.index(point.startIndex, offsetBy: 33))
        let y = point.subdata(in: point.index(point.startIndex, offsetBy: 33)..<point.index(point.startIndex, offsetBy: 65))
        return CBOR.map([
            (.int(1),  .int(2)),    // kty: EC2
            (.int(3),  .int(-7)),   // alg: ES256
            (.int(-1), .int(1)),    // crv: P-256
            (.int(-2), .bytes(x)),
            (.int(-3), .bytes(y)),
        ]).encoded()
    }
}
```

- [ ] **Step 2: Run** → FAIL.
- [ ] **Step 3: Implement**:
```swift
public enum Fido2Authenticator {
    public static func authenticatorData(rpId: String, flags: AuthenticatorFlags,
                                         signCount: UInt32, attestedCredentialData: Data? = nil) -> Data {
        var d = Data(SHA256.hash(data: Data(rpId.utf8)))
        d.append(flags.rawValue)
        d.append(contentsOf: withUnsafeBytes(of: signCount.bigEndian) { Array($0) })
        if let acd = attestedCredentialData { d.append(acd) }
        return d
    }
    public static func assert(rpId: String, clientDataHash: Data, signCount: UInt32,
                              userVerified: Bool, key: CredentialKey) throws
        -> (authenticatorData: Data, signature: Data) {
        var flags: AuthenticatorFlags = [.userPresent]
        if userVerified { flags.insert(.userVerified) }
        let authData = authenticatorData(rpId: rpId, flags: flags, signCount: signCount)
        let signature = try key.sign(authData + clientDataHash)
        return (authData, signature)
    }
}
```

### Task 4: Registration (attested credential data + attestationObject)

**Files:** extend `Sources/Fido2/AuthenticatorData.swift` + `Fido2Authenticator.swift`; `Checks_Registration.swift`.

Attested credential data = `aaguid(16) || credIdLen(2, big-endian) || credentialId || cosePublicKey`. attestationObject CBOR map = `{"fmt":"none", "attStmt":{}, "authData":<bytes>}`.

- [ ] **Step 1: Failing checks** —
  - `register(rpId: "example.com", clientDataHash: <32B>, credentialId: <16B>, userVerified: true)` → returns `(attestationObject, key)`.
  - Decode/inspect: attestationObject is a CBOR map with `fmt == "none"`, empty `attStmt` map, and an `authData` byte string. (Spot-check by encoding the expected map yourself and comparing, or by locating the `"none"` text and the authData length.)
  - The authData has the AT flag (0x40) set and length == 37 + 16 + 2 + credIdLen + len(cosePublicKey).
  - Extract the COSE public key from authData and verify it matches `COSEKey.encode(publicKeyX963: key.publicKeyX963)`.
  - Sanity: an assertion signed by the returned `key` verifies (registration→assertion handshake).
- [ ] **Step 2: Run** → FAIL.
- [ ] **Step 3: Implement**:
```swift
extension Fido2Authenticator {
    public static func register(rpId: String, clientDataHash: Data, credentialId: Data,
                                userVerified: Bool, aaguid: Data = Data(repeating: 0, count: 16))
        throws -> (attestationObject: Data, credentialKey: CredentialKey) {
        let key = CredentialKey()
        let cose = COSEKey.encode(publicKeyX963: key.publicKeyX963)
        var acd = Data()
        acd.append(aaguid)
        let len = UInt16(credentialId.count).bigEndian
        acd.append(contentsOf: withUnsafeBytes(of: len) { Array($0) })
        acd.append(credentialId)
        acd.append(cose)
        var flags: AuthenticatorFlags = [.userPresent, .attestedData]
        if userVerified { flags.insert(.userVerified) }
        let authData = authenticatorData(rpId: rpId, flags: flags, signCount: 0, attestedCredentialData: acd)
        let attObj = CBOR.map([
            (.text("fmt"), .text("none")),
            (.text("attStmt"), .map([])),
            (.text("authData"), .bytes(authData)),
        ]).encoded()
        return (attObj, key)
    }
}
```
